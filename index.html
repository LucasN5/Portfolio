GitHub Finder User

X

GitHub Finder User é um projeto para treinamento de consumo de API, o projeto utiliza a API do github para fazer requisições de busca de usuários, caso seja encontrado o usuário, é demonstrado o local onde reside, número de seguidores e quantos usuários ele segue. Todas essas informações são obtidas por meio da API do GitHub

O projeto foi desenvolvido com as seguintes tecnologias:

• HTML5

• CSS

• Typescript

• Javascript

• React

• API GitHub

• CSS

• Javascript

Lista de Afazeres

X

Lista de Afazeres é um projeto que permite ao usuário criar tarefas com a opção de definir um tipo, filtrá-las por status (concluídas, completas ou incompletas), pesquisá-las por nome, marcar tarefas como concluídas ou excluí-las, além de organizar a lista de tarefas em ordem ascendente ou descendente.

O projeto foi desenvolvido com as seguintes tecnologias:

• HTML5

• CSS

• Javascript

• React
